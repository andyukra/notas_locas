# Notas locas by andy
### Sencilla app de notas adhesivas en Flet
![Imagen Mockup 1 de Notas Locas](https://res.cloudinary.com/dwbm8g4o5/image/upload/v1771111861/notaslocas_mockup_1.jpg)
> Pantalla de inicio de Notas Locas

Mi primera aplicación en flet (adaptación de flutter en python) creada para resolver el problema de no tener donde guardar notas de forma rápida en mi teléfono

![Imagen Mockup 2 de Notas Locas](https://res.cloudinary.com/dwbm8g4o5/image/upload/v1771111861/notaslocas_mockup_2.jpg)

> Ventana modal para agregar una nueva nota

La aplicación puede realizar las siguientes funciones con notas:

* Crear
* Leer
* Editar
* Eliminar

Para editar la app se hace un git clone y luego:

```
uv sync
```
Para descargar las dependencias y crear el entorno virtual .venv
